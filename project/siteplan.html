Overview

Name

Folklore Dive - Because this site will "dive" into folklore

Purpose

The site will let users explore Brazilian folklore, with options to view detailed descriptions of different legends and save their favorites, even when they return later. A search bar and category filter will make it easy to find specific legends, while a responsive menu will help users navigate between sections. The site will be interactive, updating automatically as users browse and select legends.

Details Page

On this page, users can view detailed information about selected folklore legends. The page displays the legend's image alongside more in-depth details. Additionally, users can navigate through legends using a clickable arrow to view the next one in the list.

There are two ways to access this page:
Clicking on a legend from the main page will display the details of the selected legend.
Clicking the "Details" link in the navigation bar will take the user to this page, showing details of the first legend in the list by default.

Favorites Page

Users have the ability to add legends to their favorites from both the home and details pages. A list of all favorite legends will be displayed here. If no legends have been added to favorites, the page will display a message saying, "No legends have been saved to favorites."
